# ТНТУ Помічник 🤖

Telegram-бот для студентів Тернопільського національного технічного університету імені Івана Пулюя.

## Функції

- 📅 Розклад занять для всіх груп і факультетів (парсинг tntu.edu.ua)
- 📝 Розклад екзаменів
- 🎓 Калькулятор балів і конвертер ECTS
- 👨‍🏫 Пошук викладача
- 🌤 Погода в Тернополі
- 💵 Курс валют (НБУ)
- 📰 Новини ТНТУ
- ⏰ Особисті нагадування
- 📋 Трекер дедлайнів
- 📝 Швидкі нотатки
- ❓ Анонімний Q&A
- 🎯 Вікторина по предметах

## Запуск

```bash
pip install -r requirements.txt
export BOT_TOKEN="твій_токен"
export WEATHER_API_KEY="твій_ключ"
python main.py
```

## Стек

- Python 3.11+
- aiogram 3.x
- requests + BeautifulSoup4
- APScheduler
